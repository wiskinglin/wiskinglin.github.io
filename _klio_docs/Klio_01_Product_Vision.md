# KLIO 產品願景與概念發想

本章節記錄了初期關於本地端與 AI 應用結合的幾個核心概念想法與實作方向，作為整個專案的發想藍圖。

幾個想法
1. MD 編輯器搭配LLM的應用 產生萬字文章
    - **實作：AI_Markdown編輯器.html** (支援 Phi-3 Mini Q4 本地推理、WebGPU 加速、Jina Reader 抓取整合)
    - **實作：無限流長文產生器.html** (提供 10 種風格、動態語義矩陣技術、支援萬字串流生成)
2. TTS 聲音輸出 Chat TTS Cosy Voice, fish-speech, GPT-VoTITS
    - **實作：語音克隆體驗版.html** (整合四大模型介面、支援參考音色錄製與上傳、參數即時調整)
    - **實作：本地語音工作站_Vue版.html** (現代化 UI 介面、模擬真實工作流)
3. 影片生成 Wan Studio
    - **實作：影片生成工作站.html** (基於 Wan2.2 14B MoE 模型，支援 T2V, I2V, S2V 與角色動畫，最高 720P 輸出)
    - **開發進度：** 已建立與 FastAPI 後端通訊的串接原型
4. 檔案總管
    - **實作：智慧自動整理檔案總管.html** (SmartDrop 引擎，支援依副檔名自動分類規則、網格化視覺管理)
    - **實作：WebOS雲端檔案總管.html** (模擬桌面操作環境系統)
