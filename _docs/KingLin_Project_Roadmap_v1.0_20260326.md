# 專案規劃與發展藍圖 (Project Roadmap)

本專案旨在建構一個融合深度分析、頂尖前端技術與 AI Agent 實驗的數位平台。
核心理念：「**Think Smart. Look Amazing.**」— 結合深度商業分析與極致的 Web 前端視覺體驗。

---

## 1. 深度分析與 Gems 內容生成引擎 (Deep Research & Gems Synthesis)
**核心概念**：
將 Gemini 生成的深度商業報告（如 3C、電動車、AI 等前瞻產業），轉化為具備優美排版的「可編輯網頁摘要 (Gems)」。
**具體實作規劃**：
*   **自動化題目蒐集機制**：建立新的資料蒐集機制，有系統地匯整潛在的研究題目。
*   **Gems 網頁組件化**：LLM 產出的報告不再只是純文字。系統會將其轉化為自帶排版的網頁 UI，且允許使用者在瀏覽器中直接選取、編輯文字內容 (`contentEditable`)。
*   **無縫導出輸出**：在瀏覽器中提供一鍵功能，將這些編輯好的 Gems 摘要，完整匯出為高畫質 PDF 或獨立 HTML 檔案，便於分享與保存。

## 2. 線上主題策展：風格博物館 (Theme Curation: Style Museum)
**核心概念**：
將現有的全球 50 大主流 UX/UI 設計風格庫 (`top50/`)，轉型為可動態套用的「博物館展間模板」。
**具體實作規劃**：
*   **內容與樣式解耦**：實現一份標準的報告內容，可以隨時切換、套用不同的前衛設計風格模板。
*   **沉浸式情境體驗**：如同參觀實體博物館，讀者在閱讀相同的報告時，可根據選擇的風格（如：清晰的便當盒網格、強烈視覺衝擊的新粗野主義，或空間透視圖），獲得截然不同的氛圍感受，徹底發揮內容的魅力。

## 3. Web-Native 辦公與研究工具 (Office-Replacement Web Tools)
**核心概念**：
持續挖掘 HTML5 與 JavaScript 在瀏覽器「本地端 (Local)」的極限，最終目標是打造能取代傳統 Office 軟體的輕量化工作站。
**具體實作規劃**：
*   **現有基礎優化**：持續升級已建構的 `tools/pdf.html` (PDF 檢視/處理工具) 與 `tools/ebook.html` (電子書閱讀器)。
*   **純前端文檔編輯器**：探索 File System Access API 等技術，讓使用者無需後端伺服器，即可在本地端直接新建、編輯、儲存分析報告。打破傳統 Word 與網頁 Markdown 之間的界線。

## 4. 自動化電子書城：Smart Library (LLM-Driven Kobo Model)
**核心概念**：
打造一個介面精美、類似 Kobo 的電子書城平台，其特點在於內容完全大語言模型 (LLM) 自動編撰。
**具體實作規劃**：
*   **AI 作者群**：透過 LLM 自動針對特定主題（例如先前的投資心理學）撰寫大綱、生成深度章節內容，並自動完成排版。
*   **書城介面佈局**：結合 Top50 的優秀排版風格，建立書架介面，提供使用者瀏覽縮圖、收藏書籍與沉浸式的閱讀介面。

## 5. Agent 工作流與技能重構 (Agentic Workflow & Skills Re-planning)
**核心概念**：
因應上述四大目標，重新規劃與重構專案內的 AI Agent 機制 (Skill 與 Workflow)。
**具體實作規劃**：
*   **蒐集與研究機制**：建立專屬機制來探索並紀錄感興趣的報告題目。
*   **Gems 轉化 Workflow**：建立「發現題目 -> 深度分析 -> 產出可編輯/具排版的網頁摘要 Gems」的全自動化流程。
*   **策展與部署**：讓 Agent 具備能力，能將新報告自動掛載至最適合的博物館風格房間中，達到全自動化的內容策展。
