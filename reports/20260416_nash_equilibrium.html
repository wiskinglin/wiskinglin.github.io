純策略與混合策略推論
Nash Equilibrium Analysis

探討 3 × 3 非合作賽局中的純策略及混合策略 Nash 均衡，透過剔除嚴格劣勢策略並計算期望報酬，找出局中所有的均衡點。

這是一個 3 × 3 的非合作賽局。我們從白板上的照片可以整理出甲（列玩家）與乙（行玩家）的報酬矩陣如下：

L M R
U (1, 3) (4, 2) (3, 4)
C (5, 1) (1, 2) (2, -1)
D (3, 5) (2, 3) (2, 7)

1. 尋找純策略 Nash 均衡 (Pure Strategy Nash Equilibrium)

我們可以透過找出雙方的「最佳回應 (Best Response)」來判斷純策略均衡：

  • 甲的最佳回應（看第一個數字）：
    • 若乙選 L，甲選 C 最好 (5 > 3, 1)
    • 若乙選 M，甲選 U 最好 (4 > 2, 1)
    • 若乙選 R，甲選 U 最好 (3 > 2, 2)
  • 乙的最佳回應（看第二個數字）：
    • 若甲選 U，乙選 R 最好 (4 > 3, 2)
    • 若甲選 C，乙選 M 最好 (2 > 1, -1)
    • 若甲選 D，乙選 R 最好 (7 > 5, 3)

當雙方互為最佳回應時，即為純策略 Nash 均衡。從上面可以看出，當甲選 U 且乙選 R 時，甲的報酬 3 且乙的報酬 4 皆為該條件下的最佳選擇。
因此，純策略 Nash 均衡只有一個：(U, R)。

2. 尋找混合策略 Nash 均衡 (Mixed Strategy Nash Equilibrium)

為了計算混合策略，我們先檢查是否有嚴格劣勢策略 (Strictly Dominated Strategy) 可以剔除，這樣能大幅簡化計算。

觀察甲的策略 D，其報酬為 (3, 2, 2)，我們來測試它是否被 U (1, 4, 3) 和 C (5, 1, 2) 的某個混合策略嚴格優勢。
假設甲以機率 p = 0.4 選擇 U，以機率 1 - p = 0.6 選擇 C：

  • 對應乙選 L 的期望報酬：0.4 × 1 + 0.6 × 5 = 3.4 > 3
  • 對應乙選 M 的期望報酬：0.4 × 4 + 0.6 × 1 = 2.2 > 2
  • 對應乙選 R 的期望報酬：0.4 × 3 + 0.6 × 2 = 2.4 > 2

因為這個混合策略 (0.4U + 0.6C) 在任何情況下的報酬都嚴格大於單純選擇 D，所以 D 是嚴格劣勢策略，甲永遠不會在均衡中選擇 D。 我們可以將 D 直接刪除，得到一個 2 × 3 的簡化矩陣。

接下來，假設甲以機率 p 選 U，以機率 1-p 選 C。我們來計算乙選擇各個策略的期望報酬：

  • E(L) = 3p + 1(1 - p) = 2p + 1
  • E(M) = 2p + 2(1 - p) = 2
  • E(R) = 4p - 1(1 - p) = 5p - 1

混合策略的核心在於讓對手在不同策略間感到「無異 (Indifferent)」。我們來分情況找出這些點：

情況一：乙混合選擇 L 與 M (不選 R)
令 E(L) = E(M)：

2p + 1 = 2 ➔ 2p = 1 ➔ p = 1/2

（此時代入檢查 E(R) = 5(1/2) - 1 = 1.5 < 2，所以乙確實不會想選 R，符合假設）。
當甲的 p = 1/2 時，假設乙以機率 q 選 L，機率 1 - q 選 M。為了讓甲對 U 和 C 也感到無異，我們計算甲的期望報酬並讓其相等：

  • 甲選 U 的期望報酬：1q + 4(1 - q) = 4 - 3q
  • 甲選 C 的期望報酬：5q + 1(1 - q) = 4q + 1
4 - 3q = 4q + 1 ➔ 7q = 3 ➔ q = 3/7

這得到第一組混合策略均衡：甲的機率分佈為 (1/2, 1/2, 0)，乙的機率分佈為 (3/7, 4/7, 0)。

情況二：乙混合選擇 L 與 R (不選 M)
令 E(L) = E(R)：

2p + 1 = 5p - 1 ➔ 3p = 2 ➔ p = 2/3

（此時代入檢查 E(M) = 2 < E(L) = 7/3，所以乙確實不會想選 M，符合假設）。
當甲的 p = 2/3 時，假設乙以機率 q 選 L，機率 1 - q 選 R。為了讓甲對 U 和 C 無異：

  • 甲選 U 的期望報酬：1q + 3(1 - q) = 3 - 2q
  • 甲選 C 的期望報酬：5q + 2(1 - q) = 2 + 3q
3 - 2q = 2 + 3q ➔ 5q = 1 ➔ q = 1/5

這得到第二組混合策略均衡：甲的機率分佈為 (2/3, 1/3, 0)，乙的機率分佈為 (1/5, 0, 4/5)。

(備註：如果令 E(M) = E(R) 會得出 p = 0.6，但此時 E(L) = 2.2 > 2，乙會直接選擇 L 而不是去混合 M 與 R，因此這個情況不成立。)

總結

根據上述計算，這局賽局總共存在三個 Nash 均衡（包含一個純策略與兩個混合策略）：

  1. 純策略 Nash 均衡： 雙方選擇 (U, R)。
  2. 混合策略 Nash 均衡一： 甲以 1/2 的機率選 U、1/2 選 C；乙以 3/7 的機率選 L、4/7 選 M。
  3. 混合策略 Nash 均衡二： 甲以 2/3 的機率選 U、1/3 選 C；乙以 1/5 的機率選 L、4/5 選 R。
