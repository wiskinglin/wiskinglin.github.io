# 工業自動化、HMI 與量測儀器資源 (Industrial Automation & HMI)

本文件整理了工業用人機介面（HMI）、精密機械組件、電力電子量測儀器及自動化系統等核心工業資源，專為 PM、採購及研發工程師設計。

## 🔍 如何使用此資料
1. **硬體採購與評估**：涵蓋台達、固緯、研華及國際大廠（GE, Siemens）的工業設備。
2. **量測與測試**：包含示波器、直流電源供應器及電子負載等實驗室必備工具。
3. **組件與模組**：搜尋精密螺帽、齒輪、傳動系統及工業顯示模組供應商。
