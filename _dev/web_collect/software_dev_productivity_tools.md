# 軟體開發工作流與高生產力工具 (Dev & Productivity)

本文件整理了現代軟體開發必備的 AI 編程助手、自動化工作流、設計輔助工具及各式開發輔助平台。

## 🔍 如何使用此資料
1. **AI 配對編程**：利用 GitHub Copilot 與 Windsurf (Codeium) 提升代碼編寫速度與質量。
2. **視覺化溝通**：使用 Mermaid Live Editor 快速繪製系統架構圖與流程圖（Flowcharts）。
3. **快速修圖**：利用 remove.bg 進行 AI 智慧去背，加速簡報與網站素材準備。
