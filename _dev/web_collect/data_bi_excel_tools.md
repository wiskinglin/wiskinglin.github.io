# 數據分析、BI 工具與 Excel 進階實務 (Data BI & Excel)

本文件彙整了商業智慧（BI）軟體工具、Excel 數據分析技巧及視覺化報表設計資源，旨在提升專案管理與業務決策的效率。

## 🔍 如何使用此資料
1. **工具選型**：對比 Power BI, Tableau 及 FineReport 在企業報表開發中的優勢。
2. **Excel 進階**：學習樞紐分析表（Pivot Table）、多表合併及圖表視覺化技巧。
3. **商業智慧**：了解如何利用現金流量表與自由現金流進行財務體質分析。
