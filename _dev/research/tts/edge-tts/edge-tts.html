使用說明

🔧 一、啟動後端伺服器

  1. 確認已安裝 Python 與必要套件： 
    pip install fastapi uvicorn edge-tts
  2. 在終端機中切換至專案目錄並啟動： 
    cd c:\Dev\wiskinglin.github.io\_dev\on-going
python voice_studio_backend.py
  3. 看到以下訊息即代表啟動成功： 
    Edge TTS Voice Studio
主頁面:  http://127.0.0.1:8000
API 文件: http://127.0.0.1:8000/docs

🎙️ 二、使用語音合成

  1. 在瀏覽器中開啟 http://127.0.0.1:8000
  2. 在「文字輸入」區域輸入您想合成的內容，或點擊預設按鈕快速填入範例
  3. 系統會自動偵測語言（中文、英文、日文），無需手動選擇
  4. 選擇一個語音角色（四位角色各有不同音色表現）
  5. 調整語速（0.5x 慢速 ~ 2.0x 快速）
  6. 點擊「開始合成」按鈕
  7. 等待數秒，合成完成後可直接播放或下載 MP3 音檔

ℹ️ 三、注意事項

  1. 本工具使用 Microsoft Edge TTS 雲端引擎，需要網路連線
  2. 四位語音角色（曉晨、雲哲、曉雨、雲健）是 Edge TTS Neural 語音，非本地 AI 模型
  3. 語言偵測依據字元分布判斷：含日文假名判定日文，含中文字判定繁體中文，其餘判定英文
  4. API 文件可在 http://127.0.0.1:8000/docs 查看（Swagger UI）
  5. 生成的音檔暫存於 on-going/output/ 目錄
