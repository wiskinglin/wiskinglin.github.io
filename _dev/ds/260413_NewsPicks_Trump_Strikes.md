# トランプ氏、イランへの限定的攻撃再開を検討
- URL: https://newspicks.com/news/16432293/
- Media: NewsPicks (Reuters)
- Fetch Date: 2026-04-13

トランプ米大統領と顧問らが、停滞する和平交渉の打開策として、ホルムズ海峡の封鎖に加え、イランに対する限定的な軍事攻撃の再開を検討していることが明らかになりました。ウォール・ストリート・ジャーナルが報じました。

この動きは、外交的な停滞を軍事的な圧力で突破しようとするトランプ政権の姿勢を象徴していますが、地域のさらなる不安定化を招くとの懸念も広がっています。
