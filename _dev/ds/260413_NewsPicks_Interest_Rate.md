# 長期金利、２．４９０％に上昇
- URL: https://newspicks.com/news/16432527/
- Media: NewsPicks (JIJI)
- Fetch Date: 2026-04-13

13日の東京債券市場で、日本の長期金利の指標となる10年物国債の利回りが一時2.490%に達し、約27年ぶりの高水準を記録しました。中東情勢の緊迫化を受けた世界的な金利上昇圧力が日本にも波及しています。

金利の上昇は住宅ローンなどの借入コスト増につながる一方、運用面ではプラスの側面もありますが、株価の下落や円安の進行と相まって、日本経済への影響を注視する必要があります。
