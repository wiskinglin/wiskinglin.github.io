核心摘要 TL;DR

  • 零延遲推論：WebGPU + WASM 消弭網路傳輸瓶頸，毫秒級回應使用者意圖。
  • 極致數據隱私：敏感數據永不離端，完美符合 HIPAA、GDPR 等高規格合規要求。
  • 重塑成本結構：轉嫁算力至終端，徹底消除雲端 API Token 費用。
  • OPFS 最佳實踐：Origin Private File System 以位元組級存取處理巨量模型檔，速度為 IndexedDB 數倍。

02 / 關鍵技術矩陣

核心推理引擎

WebGPU C++ Runtime & WGSL

引入 C++ 重寫的 v4 執行階段，利用計算著色器將矩陣乘法分發至 GPU 數萬個執行緒並發執行。

PWA 離線與存儲

Background Fetch 巨量下載

將數 GB 模型下載任務移交 OS 層，應用關閉也能持續；結合 Persistent Storage 避免被當作暫存清除。

資料預處理

多模態交錯輸入與 Chat Template

前端動態將視覺張量與文字 Token 融合；內建 Jinja 模板將對話陣列精準編譯為模型專屬 Prompt。

效能調優

推測解碼 (Speculative Decoding)

前端並行草稿模型與目標大模型，透過草稿猜測並由大模型批量驗證，成倍提升生成速度。
