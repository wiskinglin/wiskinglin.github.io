# pptx 工作流

當使用者要求「將這篇文檔轉成簡報」或「幫我做一份簡報」時觸發。

1. **大綱分析**：
   - 讀取文檔內容，使用 `pptx` 的規範將其拆解為「標題」、「重點」、「補充說明」。
2. **風格選擇**：
   - 根據文檔主題，從 `pptx/SKILL.md` 的配色方案中建議 2-3 套（例如：科技文檔建議 Midnight Executive，自然議題建議 Forest & Moss）。
3. **Draft 生成**：
   // turbo
   - 使用 `pptxgenjs` 腳本生成初步簡報檔案。
4. **視覺優化**：
   - 檢查是否每一張投影片都有視覺元素（圖片、Icon、或數據大圖）。
   - 若有圖片需求，使用 `generate_image` 工具產出高品質素材並插入簡報。
5. **品質檢查 (QA)**：
   - 透過 `markitdown` 提取產出的簡報，確認文字無誤。
   - （可選，若安裝了 Poppler）生成縮圖供使用者預覽。
