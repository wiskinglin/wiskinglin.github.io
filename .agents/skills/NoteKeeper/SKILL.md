# NoteKeeper

## 任務目標

將用戶臨時輸入的文字、連結或碎片化資訊自動化「規格化」並「存檔」至 `_data/note.md` 中。確保內容易於閱讀、按周分類，且新筆記始終在最上方，方便快速回顧。

## 執行指南

### [Input]
- 用户隨手記錄的文字
- 網址連結 (URL)
- 圖片附件（若有的話，需生成描述）
- 檔案內容片段

### [Process]
1. **內容過濾與格式化**：
   - 提取內容摘要或關鍵字。
   - 若為網址，嘗試抓取網頁 `<title>` 並標記為 `[Link]` 型態。
   - 若為代碼片段，使用相應的 Markdown 代碼塊包裝。
2. **日期定位**：
   - 獲取當前台灣時間與周數 (Week Number, ISO 8601)。
   - 生成標題格式：`### YYYY-MM-DD (Day) HH:MM`。
3. **分段管理**：
   - 檢查 `_data/note.md` 中是否已存在該周的區塊標題（格式：`## YYYY-Wxx (YYYY/MM/DD - YYYY/MM/DD)`）。
   - 若不存在，則在 `_data/note.md` 最上方插入該周標題（緊接在 Markdown 文件的主標頭之後）。
4. **插入內容 (Prepend)**：
   - 將格式化後的筆記內容插入到對應周標題的正下方。

### [Output]
- 更新後的 `_data/note.md` 檔案。

## 注意事項與準則

- **Prepend 優先**：確保新周標題在最上方，新筆記在對應周標題下方。
- **連結處理**：務必檢查網址有效性，並嘗試擷取有意義的頁面標題。
- **排版風格**：保持簡約、高對比，並使用適當的 Emoji（💡, 📖, 📝, 📅）增加可讀性。
