# KLIO Lessons Learned (經驗庫)

| 日期 | QA/除錯記錄 | 預防措施與決議 |
|:---|:---|:---|
| 2026-03-27 | PDF 原生導出列印時，偶發被截斷問題 | `@media print` 必須確保添加 `page-break-after: always`，並且 `.page-container` 不可使用 `box-shadow`。 |
| 2026-03-27 | 橫式 A4 內容過少導致底部空白 | .page-container 必須確實套用 `justify-content: space-between` 填滿版面。 |
| 2026-04-13 | 每日新聞抓取產生幻覺：部分網站封阻爬蟲，導致解析到的連結與標題不符，甚至該連結內根本沒有對應的文章。 | 需評估升級抓取方案（例如加入無頭瀏覽器或使用 RSS/API），並實作防呆或內容驗證機制，確保標題、內文與連結的關聯正確性。 |
| 2026-04-13 | 每日新聞選題未如預期聚焦：目前抓取的項目看似隨機選取，未鎖定特定議題，也未能按照原定計畫抓取各來源首頁「最重要 / 最上方」的新聞。 | 需重新檢視爬取邏輯中的元素定位（CSS Selector）及選題規則，限制只抓取特定排序（如 Top 5 頭條區塊）或增加特定議題條件過濾，避免隨機撈取。 |
