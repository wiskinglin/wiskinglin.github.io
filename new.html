2026-03-31

Platform Upgrade & Mobile Experience
New Features
  • Mobile-First 閱讀體驗：全面升級行動裝置響應式設計，推出專為手機閱讀最佳化的報告介面，確保隨時皆有完美的閱讀流暢度。
  • Bento UI 視覺進化：首頁資訊板塊大幅煥新，導入高質感影像與精緻陰影層次，打造「Think Smart, Look Amazing」的高端視覺饗宴。
  • Top 50 UX/UI 策展：全新上線 50 款前衛設計風格展示庫，並新增「Curated Editorial」編輯精選 (GQ 風格) 互動原型。
  • 全新沉浸式圖書館：推出結合 Bento Grid 風格的沉浸式閱讀器與圖書館介面，供使用者探索知識。
UX / UI Imrpovement
  • 色彩無障礙優化：全面稽核並升級 KLIO Color 30 組配色系統，確保全網域所有主題設計皆符合 WCAG 對比度最高標準。
  • 閱讀無干擾設計：改善主題選擇器介面，實作隱藏滑出式選單，確保在切換深/淺主題時的視覺專注度。
  • 品牌識別統一：全站研究報告與體驗工具全面導入「KLIO x AI」品牌標準字與導覽標誌。
