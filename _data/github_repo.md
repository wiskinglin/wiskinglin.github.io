# 開源專案與開發者資源儲存庫

此檔案整理了從個人筆記中擷取出來的 GitHub 儲存庫與前端開發資源，方便快速查閱。

## AI Agent & 記憶系統
- **[anthropics/skills](https://github.com/anthropics/skills)**: Agent Skills 的公開儲存庫 (PDF, PPTX, DOCX, XLSX 等)。
- **[openclaw/openclaw](https://github.com/openclaw/openclaw)**: 你專屬的個人 AI 助理。支援任何作業系統與平台。
- **[openclaw/clawhub](https://github.com/openclaw/clawhub)**: OpenClaw 的技能目錄。
- **[HelloRuru/claude-memory-engine](https://github.com/HelloRuru/claude-memory-engine)**: 使用 hooks + markdown 打造的 Claude Code 記憶系統。

## 社群平台自動化與開發 (LINE & Instagram)
- **[Lin-jun-xiang/agent-line-bot](https://github.com/Lin-jun-xiang/agent-line-bot)**: 免費的 Agent Line Bot，具備網頁搜尋、圖片/影片生成等功能。
- **[memochou1993/gpt-ai-assistant](https://github.com/memochou1993/gpt-ai-assistant)**: OpenAI + LINE + Vercel = GPT AI 助理。
- **[line/line-bot-sdk-nodejs](https://github.com/line/line-bot-sdk-nodejs)**: 適用於 Node.js 的 LINE Messaging API SDK。
- **[InstaPy/InstaPy](https://github.com/InstaPy/InstaPy)**: Instagram 機器人 - 自動化 Instagram 互動的工具。
- **[subzeroid/instagrapi](https://github.com/subzeroid/instagrapi)**: 功能強大的 Instagram 私人 API Python 函式庫。
- **[Datalux/Osintgram](https://github.com/Datalux/Osintgram)**: 具備互動式 shell 的 Instagram OSINT (開源情報) 工具。

## 語音、音訊與影片 AI
- **[OpenMOSS/MOSS-TTS-Nano](https://github.com/OpenMOSS/MOSS-TTS-Nano)**: 多語言微型語音生成模型 (0.1B 參數)。
- **[k2-fsa/OmniVoice](https://github.com/k2-fsa/OmniVoice)**: 支援 600+ 種語言的高品質語音複製 TTS。
- **[facefusion/facefusion](https://github.com/facefusion/facefusion)**: 次世代的換臉與臉部增強工具。
- **[Wan-Video/Wan2.2](https://github.com/Wan-Video/Wan2.2)**: 開源且先進的大規模影片生成模型。

## 自動化與測試框架
- **[microsoft/playwright](https://github.com/microsoft/playwright)**: 跨 Chromium、Firefox 與 WebKit 的網頁測試與自動化框架。
- **[appium/appium](https://github.com/appium/appium)**: 適用於各類應用程式的跨平台自動化框架 (基於 W3C WebDriver)。

## 前端、UI 與設計資源
- **[Anime.js](https://animejs.com/)**: 輕量且功能強大的 JavaScript 動畫引擎，支援 CSS、SVG、DOM 屬性與 JS 物件動畫。
- **[Tailwind CSS](https://tailwindcss.com/)**: Utility-first 的 CSS 框架，用於快速構建現代化網站。
- **[DiceBear](https://www.dicebear.com/)**: 提供 30 多種風格的 SVG 頭像生成 API，適合作為用戶頭像生成器。
- **[Lorem Picsum](https://picsum.photos)**: 專為圖片提供的佔位符服務，可用於網頁原型設計。
- **[Unsplash Image CDN (Imgix)](https://images.unsplash.com)**: 支援圖片動態縮放、裁切與即時優化的 CDN 服務。
- **[Google Fonts](https://fonts.google.com/)**: 大量的開源字體庫，支援網頁直接引用。

## 雲端開發平台
- **[Firebase](https://firebase.google.com/)**: Google 的行動與 Web 應用程式開發平台，提供資料庫、認證與託管服務。
- **[OneSignal](https://onesignal.com/)**: 領先的推播通知、Email 與簡訊發送平台。
