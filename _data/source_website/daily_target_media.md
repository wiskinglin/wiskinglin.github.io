# 每日全球新聞抓取目標媒體清單

本文件為 news Skill 與對應 Workflow 專用的目標媒體清單，列出了每日早晨需自動抓取新聞的重點來源。

| 媒體類型 | 名稱 | 官方連結 | 說明 / 特色 |
| :--- | :--- | :--- | :--- |
| **數位門戶** | Yahoo! JAPAN News | [news.yahoo.co.jp](https://news.yahoo.co.jp/) | 日本流量最大的新聞集散地（門戶網站）。 |
| **新興媒體** | NewsPicks | [newspicks.com](https://newspicks.com/) | 結合深度評論功能的商業社群數位媒體。 |
| **全球電視網** | CNN | [edition.cnn.com](https://edition.cnn.com/) | 全球知名度最高的 24 小時即時新聞網。 |
| **數位平台** | Naver News | [news.naver.com](https://news.naver.com/) | 韓國民眾獲取數位新聞的最主要平台。 |
| **公共媒體標竿** | BBC (英國廣播公司) | [bbc.com/news](https://www.bbc.com/news) | 全球公共媒體與獨立報導的最高典範。 |
