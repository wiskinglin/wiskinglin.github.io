Design Token Architecture

Parametric Design Variables

The core of the Museum System is a standardized set of Design Tokens. By defining semantic variables like --theme-primary and --radius-lg, we can transform the entire visual interface without touching the underlying HTML content.

🎨 Color Harmony

Themes define primary, secondary, and background colors. Some use earthy tones (Zero-Waste), while others use high-vibrancy neons (Dopamine Colors).

📐 Layout Grids

From the strict 12-column Swiss grid to the chaotic overlapping of Anti-Design, the layout adapts to the theme's core philosophy.

Note: Content and Style are 100% decoupled. The same HTML generates radically different emotions in each theme.
